The Goal

Get more cross than the BOSS.

  Rules

The game is played on a map 1920 units wide and 1080 units high.

The spaceship have a radius of collision by 200 units.

The cross are 75 units wide and 75 units high.

  Game Input

Input for one game turn
Line 1: 1 integer x for the current x position of your spaceship.
Line 2: 1 integer y for the current y position of your spaceship.
Line 3: 1 integer nb_cross for the number of cross that you can catch.
nb_cross Line : 1 integer crossX, 1 integer crossY and 1 integer valueCross for the current coordinate (x, y) and value of the cross.
Output for one game turn
Line 1: 1 integer x , 1 integer y for the target coordinate of your spaceship and JUMP if you want to jump.
For example: 10 10 JUMP for the new coordinate x=10 y=10 and a JUMP.
(NOTE) The Y coordinate is not active for this league
Constraints
Response time for first turn ≤ 1000ms
Response time for one turn ≤ 100ms
